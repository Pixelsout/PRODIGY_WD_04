## Overview
Welcome to my personal portfolio page! This project showcases my skills, experiences, and projects in web development. The website is designed to be interactive, user-friendly, and visually appealing, reflecting my passion for web development and creative design.

## Features
Responsive Design: Ensures optimal viewing experience across all devices.
Interactive Navigation: Includes a sidebar menu that can be toggled.
Dynamic Typing Effect: Adds a dynamic typing animation for the introduction.
Modal for Project Images: Clickable project images open in a modal for an enhanced viewing experience.

## Technologies Used
* HTML5: For the basic structure of the website.
* CSS3: For styling and layout.
* JavaScript: For interactive elements like the toggle menu and typing effect.
* Remixicon: For icons used in the navigation and personal info sections.
* Boxicons: Additional icons for various sections.
## Customization
You can customize the portfolio to better fit your personal brand or preferences. Here are some areas you might want to update:

## Personal Information:

* Update your name, DOB, gender, and location in the Personal Info section of index.html.
Education and Skills:

* Modify the education and skills details in the corresponding sections of index.html.
Project Images and Details:

* Replace the project images in the images/ directory with your own project images.
Update the project descriptions in the Projects section of index.html.

## Styling:
Adjust the styles in styles.css to change the look and feel of the website.
